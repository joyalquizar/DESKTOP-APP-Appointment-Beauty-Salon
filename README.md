# DESKTOP-APP-Appointment-Beauty-Salon
Appointment Beauty Salon System is a Java desktop app built in NetBeans with MySQL and XAMPP. It features admin and client dashboards for managing appointments, payments, and services, including receipt printing and database integration for efficient salon management. 

The Appointment Beauty Salon System is a desktop-based application developed to manage salon appointments efficiently. It was created using NetBeans (Java), MySQL, and XAMPP as the local server environment.

This system is designed with both Admin and Client dashboards, allowing smooth interaction between users and salon management. Clients can book appointments, while admins can manage schedules, services, and records in an organized way.

✨ Features
🧑‍💼 Admin Dashboard for managing appointments and records
👤 Client Dashboard for booking and viewing appointments
📅 Appointment scheduling system
💳 Multiple modes of payment
🧾 Automatic receipt generation and printing
🗄️ MySQL database integration for secure data storage
🔐 User-friendly login system
🛠️ Technologies Used
Java (NetBeans IDE)
MySQL
XAMPP (Apache & MySQL Server)
🎯 Purpose

This project was developed to improve the efficiency of salon appointment management by automating booking processes, reducing manual work, and providing a smooth user experience for both clients and administrators.
